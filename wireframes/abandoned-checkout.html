Implementation notes (for the Klaviyo build)

  • Smart Sending ON.
  • One creative per email. The discount is a single Klaviyo Universal Content offer block that renders 10% (code GLOW10) for prospects and hides for customers. E3 and E4 are built once and cloned per lane with only that block toggled — no separate copywriting for the customer lane.
  • Split reads purchase history: Placed Order over all time = 0 → Prospect lane; ≥ 1 → Customer lane.
  • All gates use Placed Order, not Checkout Started (fixes the June audit finding). Every send exits if the profile places an order.
  • Entry filter includes bounce / validation suppression to fix the 13.5% Email 1 bounce.
  • E1 stays offer-free on purpose — it is the current best-performing message; the offer escalation only begins after the split.
  • SMS: straight apostrophes only, no ē macron / em dash / curly quotes; "auto-shorten links" ON so each text stays 1 credit (GSM-7).
  • E4 (Diipa): plain-text style, minimal design, reply-to a monitored inbox.
  • Cadence: Day 0 (+2h) → Day 1 → Day 2 → Day 4 → Day 6. SMS merge tags single-brace, email double-brace.
